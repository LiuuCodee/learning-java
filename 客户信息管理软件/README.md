## 项目所涉及的知识点

### Javabean

拥有 private 属性及 public 读写方法的类，成为 `Javabean`.
- 只有 `getter` 方法的属性称为只读属性
- 只有 `setter` 方法的属性称为只写属性

### this 关键字
this 关键字用于**引用当前实例**，this 是指向对象本身的指针，只能用于方法体内。
this 常用用途：
- 区分方法体中同名的属性和形参
- 区分当前对象实例的方法和其他同类对象的同名方法
